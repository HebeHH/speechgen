# Speech Generation

My general repo for speech generation projects.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.33. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.


## Used for:
- Jan 2025: Body scan meditation
